# Introduction to Genomics Data Types

This series of workshops is designed for undergraduate researchers and students who are new to working with genomic data. Some programming experience is recommended — see [Introduction to Data Science](../introDataScience/index.md) to get started. Each module builds on the previous one as you explore common genomics data types and the tools used to analyze them.

## Who This Series Is For

These workshops are designed for researchers who have completed the Introduction to Data Science series (or have equivalent R and data analysis experience) and are ready to work with real genomics data. If you can load and plot a dataset in R but have never analyzed RNA-seq output — this series is for you.


## How to Use This Series

Work through the modules in order — each one builds on skills from the previous. The course pages stay available after in-person sessions so you can revisit any section, finish exercises at your own pace, and bring any questions you may have to the CGDS Core office hours.
