# Introduction to Genomics Data Types


This series of workshops is designed for undergraduate researchers and students who are new to working with genomic data. Some programming experience is recommended - see [Intro to Data Science](../introDataScience/index.md) to get started. For this series, each module builds on the previous one as you learn about common data type in genomics and how they're analyzed.

## How to Use This Series

Work through the modules in order — each one builds on skills from the previous. The course pages stay available after in-person sessions so you can revisit any section, finish exercises at your own pace, and bring any questions you may have to the CGDS Core office hours.
