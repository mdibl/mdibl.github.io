# Spatial Transcriptomics

!!! info "Coming soon — July 2026"
    This workshop is under development and will be available for the July 2026 workshop series. Check back closer to the session date for the full course page.

Spatial transcriptomics captures gene expression while preserving the physical location of cells within a tissue section, adding the dimension of space to transcriptomic data. Technologies like 10x Visium place a tissue section on a slide covered in barcoded capture spots, so each spot's gene expression profile is tied to a set of (x, y) coordinates on the histology image.

In this workshop you will learn how spatial data is structured, how to visualize gene expression overlaid on tissue images, and how spatial patterns connect gene activity to tissue architecture and cell organization.

## Topics Covered

- How spatial transcriptomics works: capture spots, barcodes, and alignment to histology
- Data structure: spot × gene matrix with spatial coordinates
- Loading and exploring spatial data in R (Seurat)
- Visualizing gene expression on tissue sections
- Clustering spots and identifying spatially variable genes
- Comparing spatial expression patterns to histological features
- Relationship between spatial and single-cell data
